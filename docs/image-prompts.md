## HOW TO USE THESE PROMPTS

Each prompt below is ready to copy and paste directly into Nano Banana Pro (Gemini 3 Pro image model).

**Important:** Every prompt includes the Style Anchor at the end. Do not remove it. This anchor ensures all six images look like they belong to the same design system.

**Settings to use in Nano Banana Pro:**
- Model: Nano Banana Pro (not standard Nano Banana)
- Aspect ratio: 16:9 for diagram images / 4:3 for detail diagrams
- Resolution: highest available
- Public: OFF (keep private)

**If the first generation isn't right:** Don't regenerate from scratch. Use conversational editing. Say "adjust the [specific element]" and refine from there.

**Order to generate:** Create them in the order listed. Each image informs the visual language of the next.

## STYLE ANCHOR

*This block is included at the end of every prompt below. It ensures visual consistency.*

```
STYLE ANCHOR — apply to every element of this image:
Clean editorial infographic style. Background color: off-white cream (#FBF8F3). 
All text labels and connector lines use near-black ink (#131110). 
The ONLY accent colors allowed are: sage green (#8FAF94 — use for GREEN / open signals), 
terracotta red (#E05C34 — use for RED / resistant signals), and 
mustard yellow (#E9B840 — use for YELLOW / drifting signals). 
No other colors. No gradients. No drop shadows. No decorative textures. 
Typography: clean modern sans-serif (similar to IBM Plex Sans or Geist) for all labels 
and data; use an elegant serif italic (similar to Playfair Display) only for any 
major display text or section titles. 
Layout: IBM Carbon design system aesthetic — grid-aligned, structured, 
generous whitespace, 8px spacing unit. 
Node shapes: clean rectangles or rounded rectangles with thin ink borders (1–1.5px). 
Connector lines: 1.5px ink, with small arrowheads. 
Overall feel: serious, editorial, professional. A premium reference tool, not a 
startup marketing asset.
```
